Plugin API: onGetInfo
Boolean onGetInfo ( void *gp, EventPtr eventP );

Description
If non-void, allows an About dialog to be displayed for the plugin. This function is a standard form event handler.
Paramaters
*gp A pointer to the plugin global variables.
eventP Event data.

Result
Returns true if the event was handled, false if the event is to be passed through to the system.

Comments
When the ? button is pressed next to the plugin's name, form 2000 from the plugin is loaded and this function is specified as the event handler.
