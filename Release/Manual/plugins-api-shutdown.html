Plugin API: PlugInShutdown
UInt32 PlugInShutdown ( EntryPoints *ep, void *gp, UInt16 stage );

Description
This function is called when the application exits, or your plugin is unchecked in the plugin list. This function is passed a list of functions (entry points, *ep) that it can assign to the plugin's internal functions.
Paramaters
*ep The EntryPoints structure contains a list of pointers to functions - the plugin assigns internal functions to these pointers so FileCaddy knows what, and how, to call the handlers.
*gp This is a generic pointer to a chunk of memory the plugin can use to store it's global variables. This argument is passed to every function in the plugin.
stage Currently unused, this variable would allow the plugin to know if it's in early startup, late startup, or demand startup.

Result
The function returns a PalmOS error code, or 0 otherwise. Extra error codes are defined in the plugin.h file.

Comments
This function is required (and must be first) for the plugin to function.
