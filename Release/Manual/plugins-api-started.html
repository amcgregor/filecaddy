Plugin API: Getting Started
A plugin is a single code-resource application, with a minimum of three functions. The following should be included in the resource:


code0The main code resource. Required.
tAIB1000Large icon.
tAIB1001Small icon (displayed in list).
tAIN1000Plugin name (displayed in list).
tFRM1000Preferences dialog.
tFRM2000About dialog.
tSTR1000A short description/copyright message.
