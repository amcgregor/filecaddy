Plugin API: onChangeVolume
UInt32 onChangeVolume ( void *gp, UInt16 old, UInt16 new );

Description
If non-void, this function would be notified every time the user selects a volume (other then the current volume) from the dropdown list.
Paramaters
*gp A pointer to the plugin global variables.
old The previous VFS volume identifier.
new The new VFS volume identifier.

Result
The function returns a PalmOS error code, or 0 otherwise. Extra error codes are defined in the plugin.h file.

Comments
Though I have no idea what use this function could be to a developer, I have included it anyway.
