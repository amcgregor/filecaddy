Plugin API: onMenuTap
UInt32 onMenuTap ( void *gp, UInt16 vol, Char **f );

Description
If non-void, displays a menuitem in the popup list when a user holds the stylus down on an item. This function is run when the user has tapped a file.
Paramaters
*gp A pointer to the plugin global variables.
vol The VFS volume identifier.
**f An array of selected filenames (including full path).

Result
The function returns a PalmOS error code, or 0 otherwise. Extra error codes are defined in the plugin.h file.

Comments
This is the main function for any file-handler style plugins. The plugin's name is loaded from the tAIN resource.
