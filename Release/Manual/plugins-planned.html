Planned Plugins
The following plugins are planned: (you can find the entire list online at http://filecaddy.sourceforge.net)

ID3InfoMP3
View and modify MP3 ID3 tags. ID3 tags are used to display information like author, title, album, and genre. Both version 1 and version 2 are planned (plugin versions 1.0 and 1.1).
PropertiesAll Files
Show extended properties about the selected file. All attributes will be settable. Seperate dialogs for databases and files.
BeamAll Files
Beam any file to another handheld (database or file). Only one at a time, though. (Which should be improved in a later version.)
Quick ViewAll Files
View text files (html/rtf files as well), view palm databases by record, and examine resources. View only, though.
FTP Server...Network
VFS enhancement to connect to FTP or SFTP servers.
Samba Server...Network
VFS enhancement to connect to Windows style Samba shares.
NFS Server...Network
VFS enhancement to connect to UN*X style NFS shares.
Zip File...Compressed Files
Add, copy, remove, and explore files in a .zip, .tar, or .tar.gz archive, just like they were uncompressed.
Website...Network
Download files from a website.
FiletypeAll Files
Discover just exactly what type of file is selected. Requires a "magic" file.
RunPalm Files
Run Palm applications, or open databases, from VFS.
