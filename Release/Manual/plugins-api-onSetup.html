Plugin API: onSetup
Boolean onSetup ( void *gp, EventPtr eventP );

Description
If non-void, allows display of a setup dialog for the plugin. Form 1000 is created and opened, and this function is used as the event handler.
Paramaters
*gp A pointer to the plugin global variables.
eventP Pointer to event data.

Result
Returns true if the event was handled by the event handler, false otherwise, or if you want the event passed through.

Comments
This function should be structured as a normal form event handler, save the use of *gp for global variables.
