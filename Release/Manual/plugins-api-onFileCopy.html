Plugin API: onFileCopy
UInt32 onFileCopy ( void *gp, UInt16 src, UInt16 dest, Char **f, Boolean done );

Description
This function is used to interrupt or log the transfer of files.
Paramaters
*gp A pointer to the plugin global variables.
src Source VFS volume identifier.
dest Destination VFS volume identifier.
**f Source and destination filenames (including path, if any).
done Has the operation already been completed?

Result
The function returns a PalmOS error code, or 0 otherwise. Extra error codes are defined in the plugin.h file.

Comments
This function is called twice - once before the file is copied, and once after. This allows a plugin to cancel the transfer of a file (say, a restricted datafile) before it has begun.
