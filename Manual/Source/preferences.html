Preferences
The preferences dialog gives you a list of many options to select:

Bold Directories
Directories are displayed with bold text.
Sort on Startup
Enables sorting for those who want just that extra bit of speed.
Show Icons
Allows you to hide file/application icons. Speeds up display when disabled.
Show ROM Databases
When viewing internal memory, also include databases in ROM (read-only memory).
Color-coded Attributes
Not funtional yet, will eventually allow you to determine read-only, hidden, etc, files using color.
Show Hidden Files
Display hidden databases and files.
Show Directories First
When sorting, sort directories to the top of the list.
Reverse Sort Order
Sort descending in place of ascending. This can be done automatically by tapping the column headers.
Only Show Palm Files
Show Palm files (prc, pdb, pqa) on expantion cards only. You will not be able to manipulate non-palm files with this enabled.
Form Layout
Allows you to have the scrollbar on the left or right sides.
