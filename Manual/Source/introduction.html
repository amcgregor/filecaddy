FileCaddyBuild Number: 1159

Introduction
FileCaddy is a general purpose file management application for VFS enabled PalmOS devices. Compatible devices include SD slot equipped Palm handhelds, Sony Clie's, Handera handhelds, SD slot equipped Handspring Treos, and Handspring Visors with FlashAdapter or MemPlug Springboard modules.

Filecaddy will eventually have plugin support (it is quite experimental right now) with a very powerful API. This would allow developers to create file handlers (plugins that react to files), or plugins that give you more volumes. For example, a plugin that views and modifies MP3 ID3 tags, or a plugin that allows you to connect to a FTP server.

The name, FileCaddy, was picked from the many suggested in May of last year. FileCaddy was Frank Hamrick's suggestion... thanks Frank!
