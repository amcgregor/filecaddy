Menus
There are three main menus, Volume, Options, and Help.

Volume
Create Directory...
Allows you to create a directory on an expantion card.
Refresh View
If files have changed, but the display hasn't been updated, you can do it by hand.
Find Files...
Not yet implemented, you will eventually be able to search for files on your handheld or expantion cards.
Volume Information...
Displays information about the expantion card: label, media type, free/used/total space, and formatting options.

Options
Preferences...
Change many of the most common options using this dialog.
Sort...
(currently disabled)
Plugins...
(currently hidden)

Help
Release Notes...
Displays any notes about the version of FileCaddy you are running. This includes new features, bugfixes, and general comments.
About FileCaddy...
Displays the about dialog, showing copyright information and the current build number.
