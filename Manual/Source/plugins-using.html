Using Plugins
Before you can use a plugin, you have to install and activate it. Plugins are PRC files, ususally included with the FileCaddy package you downlaoded. Some plugins (including ones more recent then when you downloaded) can be found at http://filecaddy.sourceforge.net.

Once you have installed the plugins you want in whatever way is most comfortable for you, launch FileCaddy, tap the Menu silkscreen button, and go to the Options menu and tap "Plugins...". There should be a list of all installed plugins with checkboxes next to them. Check the ones you wish to enable.

Once enabled, plugins can register themselves in one of two ways: as a file handler (you tap and hold on a file, it pops up a menu of plugins), or as a VFS extention (appears under the volume drop-down). There is no need to exit the application and restart it to start using the plugins.
