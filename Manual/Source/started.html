Getting Started
The main screen is fairly easy to understand and has the following interesting areas:
Volume Dropdown
Right Column Selector
File List Item
Current Directory
Command Buttons

To get started copying files between internal memory and an expantion card, use the Volume Dropdown to pick your card (usually the second item in the list) and tap on directory names to move around. Once you are in the directory you want the files to be copied to (as seen in the Current Directory display), switch back to Internal memory, select the files by tapping on them or using the Select button, and Copy or Move.

Volume Dropdown
The dropdown list contains at least one item, "Internal". This represents your handheld's RAM. For each expantion card you have in your device when FileCaddy is run (most devices only have one, but a few have two or more) there will be an item with the card's volume name. Also in the list would be any volumes added by any plugins you have active. An example of an expantion plugin would be an FTP client.

Right Column Selector
This dropdown list contains all the different information you can display in the right column. Currently, you can display file size, PalmOS file type (i.e. REAd), or the PalmOS file creator (i.e. TEXt).

File List Item
Each line represents one database, file, or folder. If you tap on a file or database, it will be selected (checkbox), but if you tap on the icon or name of a folder, you will be moved into that folder. You can select folders, if you tap on the checkbox. You can hide the icons, or stop folders from being bold in the Preferences... screen.

Current directory
This shows what directory you are currently in. When you copy files from RAM to a card, they will be copied to the directory that is displayed here.

Command Buttons
These buttons allow you to quickly copy, move, or delete the selected files. The "Select" button pops up a list allowing you to choose to select all files, select no files, or invert the selection (all checked items are unchecked, and unchecked items become checked).
